Thesis on Road Detection in Images Using a Probablistic Graphical Model

This project made use of Python and C++. Superpixels were used to segment images taken by a car-mounted camera, to reduce the processing power required to identify where the road is in the images. A logistic regression model was trained to predict whether a superpixel consists of road pixels based on the superpixel's colour characteristics. The Canny edge detector was used to obtain the "edges" between neighbouring superpixels. The logistic regression model predictions and information about "edges" between neighbouring superpixels were then combined using a PGM to infer whether a superpixel is road based on these uncertain characteristics. The whole thesis can be found on Github.
